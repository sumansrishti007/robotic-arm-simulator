An interactive web-based simulator for a 6 degree-of-freedom robotic arm with forward kinematics, inverse kinematics, and trajectory planning capabilities.



\## Features



\- \*\*3D Visualization\*\*: Real-time 3D rendering using Three.js

\- \*\*Forward Kinematics\*\*: Calculate end-effector position from joint angles

\- \*\*Inverse Kinematics\*\*: Solve joint angles for target positions

\- \*\*Trajectory Planning\*\*: Smooth trajectory generation and animation

\- \*\*Interactive Controls\*\*: Real-time joint manipulation via sliders

\- \*\*Modern UI\*\*: Material-UI components with responsive design
